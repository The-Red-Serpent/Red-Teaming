## Large Language Model

A Large Language Model is a decoder-only Transformer trained on a massive corpus of text with a single objective  predict the next token. The "large" refers to three things simultaneously: large number of parameters (billions to trillions), large training dataset (trillions of tokens), and large compute used for training (thousands of GPUs for months). The architecture is the same Transformer decoder you already know  what changes is scale, training data, and the training pipeline applied on top.

The distinction between a Transformer and an LLM is not architectural  it is about scale and training. A base Transformer trained on a small dataset to translate French to English is not an LLM. GPT-3 trained on 300 billion tokens with 175 billion parameters is. The emergent capabilities  reasoning, instruction following, code generation, in-context learning  appear only at sufficient scale. Below a threshold, the model just pattern matches. Above it, qualitatively different behavior emerges.
